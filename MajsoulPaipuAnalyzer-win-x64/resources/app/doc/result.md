# 统计方式、名称意义、特殊情况的部分说明

双响在铳了局数中算铳了两局；三响如果流局了不算铳（雀魂不存在）

双响铳了5200+7700，不把铳点相加仅分开计数

和了收入指包含本场供托；包括自己放上去的自己的立直棒

素点即扣了本场和供托

AL相关以刚进入AL时的数据为基准；北起AL4位连庄+西入飞人是避四+逆一

副露相关由于要分副露等级目前待实现

默听指门清未立听牌

暗杠不算副露

副露和了指和了的局数中副露了的局数占比；非与总副露局数比

中途飞人视为AL位序不变

炸庄指坐庄被自摸

里宝和一发率均考虑立直和牌下概率

流局局数不包括途中流局和流局满贯

流局点数收支不包括流局满贯收支
